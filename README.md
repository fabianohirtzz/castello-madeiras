# Castello Casas de Madeira — Protótipo de site

Landing page (protótipo) para a Castello Casas de Madeira, fabricante de casas de madeira
chave na mão em Tubarão, Santa Catarina.

Site estático em HTML, CSS e JavaScript, sem build step.

- `index.html` — página única
- `css/style.css` — estilos (design system Castello)
- `js/main.js` — interações (nav, menu mobile, reveal no scroll, parallax do hero)
- `images/` — logos da marca
- `fotos-casas/` — fotografias reais de casas entregues

> Protótipo com `noindex` para apresentação. Conteúdo de depoimentos é representativo e será
> substituído pelos reviews reais antes da produção.

Desenvolvido por Freela In Home.
